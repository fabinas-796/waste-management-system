# ECOFY — Online Waste Management & Collection Platform

ECOFY is a full-stack waste management platform that allows users to book waste collection slots, request pickups via photo submission, and earn reward points for sustainable participation. The platform was built as a collaborative project with **Haritha Karma Sena (HKS)**, a government-affiliated environmental body, to streamline community-level waste collection and management.

## 🌱 Overview

ECOFY aims to make waste management effortless for both residents and waste collection workers by combining a user-friendly booking system, a reward-based incentive model, and direct coordination with HKS officers. The platform also integrates educational resources to encourage eco-friendly habits.

## ⚙️ Tech Stack

- **Backend / Web App:** Python (Flask)
- **Mobile App:** Android (Android Studio)
- **Database:** MySQL

## 🧩 Core Modules

- **Admin** — Platform oversight and management
- **IT Officer** — Technical/system administration support
- **User** — Book waste collection slots, submit pickup photo requests, track and redeem reward points
- **HKS (Haritha Karma Sena)** — Receive and act on collection requests from users

## 🚀 Key Features

- 📅 **Slot-based Waste Collection Booking** — Users can schedule collection on a preferred date
- 📸 **Photo-based Pickup Requests** — Users submit photos of waste for HKS officers to review and collect
- 🏆 **Reward Points System** — Incentivizes active participation in waste management initiatives
- 📚 **Educational Resources** — In-app content promoting waste reduction and recycling practices
